## 九万助手前端代码规范

### 1. 为什么要有团队代码规范？  
虽然这些细节是小事，不会有体验或者性能上的优化，但是却体现了一个coder和团队的专业程度，所以不管团队有多少人，代码风格都应该师出同门！ 

### 2. 如何使用？ 
在使用之前花一点时间把[规范](//awayjin.github.io/codeguide/guide/)看一遍是很必要的，
